# MarketPlace Backend



- [Express](http://expressjs.com/) and [Mongoose](http://mongoosejs.com/) for the backend
- [Webpack](https://webpack.github.io/) for compilation


## Requirements

- [Node.js](https://nodejs.org/en/) 6+

```shell
npm install
```




## Running

Development (Webpack dev server) mode:

```shell
npm run start:dev
```
